# Core Java Concepts

This repository contains well-structured and practical examples of **Core Java concepts**, created to strengthen fundamental understanding and support interview preparation.

The code focuses on clarity, correct usage of APIs, and real-world relevance rather than just theory.

---

This repository contains a structured collection of Core Java programs created to practice and demonstrate fundamental Java concepts and Object-Oriented Programming (OOP) principles.

## Topics Covered
- Encapsulation with validation
- Inheritance
- Polymorphism (method overloading & overriding)
- Abstraction (abstract classes & interfaces)
- Constructors
- Collections framework
- Exception handling
- String handling
- File handling
- multithreading

## Purpose
- Strengthen Core Java fundamentals
- Practice clean and readable code
- Build a strong foundation for backend development (JDBC, Servlets, Spring)

## Technologies
- Java (Core Java)
- Eclipse IDE
